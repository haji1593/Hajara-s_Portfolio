## Overview

A team project to build a data system for mental-health facilities — the
unglamorous but essential groundwork that good data science depends on.

## What I did

- Designed and built a database to **collect and manage** facility data.
- Integrated **multiple data sources** into one consistent store.
- Enforced **data quality, consistency, and structured storage** so downstream
  analysis could trust the inputs.

## Why it matters

Models are only as good as the data feeding them. This project is the half of
data work that happens before the modeling — and the half that decides whether
the modeling is worth anything.
